# Pokechema

Este proyecto es una aplicación web construida con [Angular](https://angular.dev/).

## Requisitos Previos

Asegúrate de tener instalado en tu sistema:
- [Node.js](https://nodejs.org/) (versión recomendada LTS)
- NPM (viene incluido con Node.js)
- [Angular CLI](https://angular.dev/tools/cli) (opcional a nivel global, pero recomendado)

## Cómo iniciar el proyecto

Sigue estos pasos para ejecutar el proyecto en tu máquina local:

1. **Navegar al directorio del proyecto:**
   ```bash
   cd Pokechema
   ```

2. **Instalar las dependencias:**
   Ejecuta el siguiente comando para descargar e instalar todos los paquetes necesarios definidos en el archivo `package.json`:
   ```bash
   npm install
   ```

3. **Iniciar el servidor de desarrollo:**
   Una vez instaladas las dependencias, levanta el servidor local de Angular ejecutando:
   ```bash
   npm start
   ```
   *Alternativamente, si tienes Angular CLI instalado de forma global, puedes usar: `ng serve`*

4. **Ver la aplicación en el navegador:**
   Una vez que el servidor esté corriendo, abre tu navegador web y dirígete a:
   [http://localhost:4200/](http://localhost:4200/)

La aplicación se recargará automáticamente en el navegador cada vez que modifiques y guardes algún archivo del código fuente.
